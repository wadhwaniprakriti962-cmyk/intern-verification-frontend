# Intern Verification System – Frontend

Frontend application for the Intern Verification System.  
This application provides an interactive and secure interface for HR authentication, intern management, and verification processes.

---

## 🚀 Live Demo

Frontend Deployment:  
https://intern-verification-frontend.vercel.app

---

## 📌 Project Overview

The frontend is developed using React.js and Vite.  
It allows HR users to:

- Register and Login securely
- Access protected dashboard pages
- View intern records
- Verify intern credentials
- Interact with backend APIs

---

## ✨ Features

- Modern Responsive UI
- HR Registration & Login
- JWT Authentication Integration
- Protected Dashboard Access
- Intern Verification Interface
- API Integration using Axios
- Logout Functionality
- React Router Navigation
- Environment Variable Support
- Deployment on Vercel

---
